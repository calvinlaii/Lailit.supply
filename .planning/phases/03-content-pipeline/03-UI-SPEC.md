## Spacing Scale

Multiples of 4px only. Consistent with Phase 1 and Phase 2.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge internal padding, inline chip gap |
| sm | 8px | Card internal gap (data-size=sm), chip row gap between chips |
| md | 16px | Card padding (px-4), section horizontal padding, filter row padding |
| lg | 24px | Card gap in grid, section vertical padding |
| xl | 32px | Page section top/bottom margin |
| 2xl | 48px | Major section breaks (e.g. explore page hero-to-grid gap) |
| 3xl | 64px | Page-level top padding on /explore |

Exceptions:
- TopNav height: 64px (h-16) — inherited from Phase 1, do not change
- Touch targets: 44px minimum height for all interactive elements (chip filters, copy button, format tabs) — inherited from Phase 1 mobile-first constraint
- Grid gap: 16px (gap-4) mobile, 24px (gap-6) tablet+

## Color

Light mode only (no dark mode in Phase 3). Uses existing shadcn CSS variables from globals.css.

| Role | CSS Variable | OKLCH Value | Usage |
|------|-------------|-------------|-------|
| Dominant (60%) | --background | oklch(1 0 0) — white | Page background, card surface, TopNav bg |
| Secondary (30%) | --muted | oklch(0.97 0 0) — neutral-50 | Thumbnail placeholder bg, card footer bg, filter row bg |
| Accent (10%) | --primary | oklch(0.205 0 0) — neutral-950 | CTA buttons, "Gratis" badge bg, copy button |
| Destructive | --destructive | oklch(0.577 0.245 27.325) — red | Not used in Phase 3 |

Accent (`--primary` / neutral-950) is reserved for:
1. "Coba Gratis" / "Jelajahi" CTA button in /explore hero
2. "Gratis" (Free) badge background on resource cards
3. Copy button background on resource detail pages
4. Active/selected state on category filter chips

Secondary semantic color for "Premium" badge: `--secondary` (oklch(0.97 0 0) bg, neutral-950 border) with a Lock icon — visually distinct from "Gratis" badge without introducing a new color.

Paywall placeholder surface: `--muted` background with `--border` (oklch(0.922 0 0)) outline.

Category placeholder thumbnail gradients (one per category, Phase 3 placeholder only):
- `animation`: linear-gradient from neutral-200 to neutral-300 (cool gray)
- `ui-components`: linear-gradient from neutral-100 to neutral-200
- `layout`: linear-gradient from neutral-200 to neutral-400 (medium gray)
- `interactions`: linear-gradient from neutral-100 to neutral-300
All gradients use only the existing neutral scale — no new color tokens.

## Interaction Contract

### Category Filter Chips

- State: selected chip uses `bg-primary text-primary-foreground` (neutral-950 bg, white text)
- Unselected: `border-border text-foreground bg-background hover:bg-muted`
- Transition: `transition-colors duration-100`
- Touch target: min-height 44px (py-2.5 at 14px text = ~40px + border = use min-h-[44px] explicitly)
- Filter is client-side — no URL change, no page reload (D-06)
- ARIA: `role="tablist"` on row, `role="tab"` on each chip, `aria-selected` on active chip

### Format Tabs

- Only render tabs listed in `available_formats` frontmatter — absent formats are not rendered at all (D-10)
- Active tab: bottom border `border-b-2 border-primary` style
- Inactive tab: `text-muted-foreground hover:text-foreground`
- Tab labels: "Framer", "Webflow", "HTML", "JSX", "TSX" (exact platform names per CONTEXT.md specifics)
- ARIA: `role="tablist"` + `role="tab"` + `aria-selected` + corresponding `role="tabpanel"`
- Touch target: min-height 44px

### Copy Button

- Default state: Copy icon (lucide `Copy`, 16px), label "Salin" visually hidden (sr-only)
- Success state: Check icon (lucide `Check`, 16px, text-green-600), reverts to Copy after 2000ms
- Position: top-right corner of CodeBlock, absolute-positioned
- Size: 32px × 32px icon button (size="icon" from Button component)
- ARIA: `aria-label="Salin kode"` (default), `aria-label="Tersalin"` (success)

### Premium Paywall Stub (Phase 3)

- Server returns 403 for `is_premium: true` resources to unauthenticated requests (D-09)
- Visual: `PaywallStub` component rendered in place of CodeBlock
- Layout: card with `bg-muted/50 border border-border rounded-xl p-8` — centered content
- Lock icon: lucide `Lock`, 24px, `text-muted-foreground`
- Heading: "Konten Premium" (16px, semibold)
- Body: "Berlangganan untuk mengakses semua format kode premium." (14px, regular, text-muted-foreground)
- CTA button: "Lihat Paket Harga" → href="/pricing" (Button variant="default", full-width on mobile)
- No blur effect in Phase 3 — full opacity placeholder (blur paywall arrives in Phase 5 with real auth)

### Empty State (Category Filter yields 0 results)

- Not expected with 20–30 seed resources across 4 categories, but must handle gracefully
- Layout: centered in grid area, `py-16 text-center`
- Icon: lucide `Search`, 32px, `text-muted-foreground`
- Heading: "Tidak ada komponen" (16px, semibold)
- Body: "Coba pilih kategori lain." (14px, regular, text-muted-foreground)

## Accessibility Contract

Inherited from Phase 1 baseline. Phase 3 additions:

| Pattern | Requirement |
|---------|-------------|
| ResourceCard | `<article>` element with `aria-label="{title}"` |
| CategoryFilterRow | `role="tablist"`, chips use `role="tab"` + `aria-selected` |
| FormatTabBar | `role="tablist"` + `role="tab"` + `aria-selected` + `role="tabpanel"` with `aria-labelledby` |
| CodeBlock | `<pre><code>` with `lang` attribute; Copy button has `aria-label` |
| ThumbnailPlaceholder | `role="img"` + `aria-label="{title} thumbnail"` |
| VideoPlaceholder | `role="img"` + `aria-label="Pratinjau video belum tersedia"` |
| PaywallStub | No `aria-hidden` — content is meaningful (upgrade path) |
| Premium badge | Includes Lock icon with `aria-hidden="true"` — text "Premium" is the accessible label |
| Focus ring | `focus-visible:ring-2 focus-visible:ring-ring/50` (inherited from shadcn base) |
| Touch targets | 44px minimum for all interactive elements |
| Reduced motion | `@media (prefers-reduced-motion: reduce)` already in globals.css — no new animations in Phase 3 |
